# Linux Commands Practice Environment

Welcome to the Linux Commands practice environment! This directory contains guides, exercises, and resources to help you master essential Linux commands.

## 📚 Contents

- **guides/** - Detailed guides for various Linux command categories
- **exercises/** - Hands-on practice exercises with solutions
- **cheat-sheets/** - Quick reference guides
- **scripts/** - Sample shell scripts for practice

## 🎯 Learning Path

### Beginner Level
1. File and Directory Operations
2. File Viewing and Searching
3. File Permissions and Ownership
4. Text Processing Basics

### Intermediate Level
5. System Information and Monitoring
6. Process Management
7. User and Group Management
8. Networking Commands

### Advanced Level
9. Scripting Basics
10. Advanced Text Processing
11. System Administration
12. Performance Tuning

## ✅ How to Use This Repository

1. Start with a topic from the Learning Path
2. Read the guide in the `guides/` directory
3. Practice the exercises in `exercises/`
4. Refer to cheat sheets when needed
5. Experiment with scripts and modify them

## 🔍 Quick Command Categories

- **File Management**: `ls`, `cd`, `mkdir`, `cp`, `mv`, `rm`
- **File Viewing**: `cat`, `less`, `more`, `head`, `tail`
- **Search & Filter**: `grep`, `find`, `locate`
- **Permissions**: `chmod`, `chown`, `umask`
- **System Info**: `uname`, `whoami`, `df`, `du`, `top`, `ps`
- **Text Processing**: `sed`, `awk`, `cut`, `sort`, `uniq`
- **Networking**: `ping`, `ifconfig`, `netstat`, `ssh`, `scp`

## 💡 Tips for Success

- Practice regularly (15-30 minutes daily is better than cramming)
- Use `man` command to read manual pages: `man command_name`
- Create test files and directories to practice safely
- Try to combine commands using pipes (`|`)
- Keep notes on commands you find useful
